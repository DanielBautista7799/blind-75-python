## Problem Summary

Given a list of integers, determine whether any value appears more than once.

This summary is intentionally brief and written in my own words. The linked practice page contains
the complete statement and constraints.

## Inputs and Expected Output

- Input: a list of integers.
- Output: `True` when at least one duplicate exists; otherwise `False`.

## Edge Cases to Consider

- Empty input.
- A one-element list.
- Every value is unique.
- Several values are duplicated.
- Negative numbers.
- Repeated zeroes.
