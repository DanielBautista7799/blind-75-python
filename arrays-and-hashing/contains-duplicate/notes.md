## First Thoughts
This problem was nothing too complez there was definitely many approaches to take to complete it but for it to be the most efficent we would want to use a hashmap or a hashset.  We definitely just want to iterate through every number and see if we have seen it before.

## Brute-Force Approach
Brute-force was not necessary but would probably take too much time. It would likely have to go through each number and iterate through the whole list.

## Optimized Approach
The hashmap or set is best for this as it completes all operations in 0(1) and for to search for a specific value does the same making the approach take 0(n) time.

Hashset

0(n)
