## Problem Summary

Given a list of strings, group together the words that are anagrams of each other.

Words are anagrams when they contain the same characters with the same frequencies, even if the letters are in a different order.

## Inputs and Expected Output

- Input: a list of strings.
- Output: a list of groups, where each group contains words that are anagrams of each other.

## Edge Cases to Consider

- Empty input list.
- A string that is empty.
- Multiple words that belong to the same anagram group.
- Words that do not have any matching anagrams.
- Duplicate words.
- Single-character strings.
