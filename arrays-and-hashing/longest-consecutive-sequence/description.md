## Problem Summary

Given an unsorted list of integers, find the length of the longest sequence of consecutive numbers.

The numbers do not need to be next to each other in the original list.

## Inputs and Expected Output

- Input: a list of integers.
- Output: an integer representing the length of the longest consecutive sequence.

## Edge Cases to Consider

- Empty list.
- One number.
- Duplicate numbers.
- Negative numbers.
- Sequence appears out of order.
- Multiple separate sequences.
