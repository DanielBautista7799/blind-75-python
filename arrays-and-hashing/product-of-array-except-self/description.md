## Problem Summary

Given an integer array, return a new array where each index contains the product of every number except the number at that index.

The solution should not use division.

## Inputs and Expected Output

- Input: a list of integers.
- Output: a list where each position stores the product of all other numbers.

## Edge Cases to Consider

- The array contains zero.
- The array contains more than one zero.
- Negative numbers.
- All positive numbers.
- Small arrays.
