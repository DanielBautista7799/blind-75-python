## Problem Summary

Given a list of integers and a number `k`, return the `k` most frequent elements from the list.

The order of the returned elements does not matter.

## Inputs and Expected Output

- Input: a list of integers `nums` and an integer `k`.
- Output: a list containing the `k` numbers that appear most frequently.

## Edge Cases to Consider

- Only one number in the list.
- `k` is equal to 1.
- `k` is equal to the number of unique values.
- Negative numbers.
- Multiple repeated values.
- Different numbers with close frequencies.
