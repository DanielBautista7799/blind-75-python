## Problem Summary

Given two strings, determine whether one string is an anagram of the other.

An anagram contains the same characters with the same frequencies, but the characters may appear
in a different order.

## Inputs and Expected Output

- Input: two strings, `s` and `t`.
- Output: `True` when the strings are anagrams; otherwise `False`.

## Edge Cases to Consider

- The strings have different lengths.
- Both strings are empty.
- The strings contain the same letters in a different order.
- One character appears a different number of times.
- The strings are already identical.
