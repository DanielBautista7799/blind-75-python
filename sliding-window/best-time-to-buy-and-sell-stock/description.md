## Problem Summary

Given a list of stock prices by day, find the maximum profit possible from buying once and selling once later.

The sell day must come after the buy day.

## Inputs and Expected Output

- Input: a list of integers representing prices.
- Output: the maximum possible profit.

## Edge Cases to Consider

- Prices only go down.
- Prices only go up.
- Best buy price is at the beginning.
- Best sell price is at the end.
- Multiple repeated prices.
- Only one possible transaction.
