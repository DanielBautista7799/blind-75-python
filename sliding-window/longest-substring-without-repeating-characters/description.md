## Problem Summary

Given a string, find the length of the longest substring that does not contain any repeated characters.

A substring must be continuous.

## Inputs and Expected Output

- Input: a string.
- Output: an integer representing the length of the longest substring with no duplicate characters.

## Edge Cases to Consider

- Empty string.
- One character.
- All characters are the same.
- No repeated characters.
- Repeated character appears after a valid window.
- Spaces or symbols inside the string.
