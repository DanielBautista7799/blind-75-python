## Problem Summary

Given a list of integers, return all unique triplets where the three numbers add up to zero.

The solution should not return duplicate triplets.

## Inputs and Expected Output

- Input: a list of integers.
- Output: a list of unique triplets where each triplet sums to zero.

## Edge Cases to Consider

- Empty list.
- Less than three numbers.
- No valid triplets.
- Duplicate values.
- Multiple duplicate triplets.
- All zeroes.
- Negative and positive numbers mixed together.
