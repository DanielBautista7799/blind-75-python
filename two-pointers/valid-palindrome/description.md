## Problem Summary

Given a string, determine whether it is a palindrome after ignoring uppercase letters, spaces, punctuation, and other non-alphanumeric characters.

A palindrome reads the same forward and backward.

## Inputs and Expected Output

- Input: a string.
- Output: `True` if the cleaned string is a palindrome, otherwise `False`.

## Edge Cases to Consider

- Empty string after cleaning.
- String with only spaces or punctuation.
- Uppercase and lowercase letters.
- Numbers mixed with letters.
- Normal palindrome phrase.
- Non-palindrome string.
